Web-based flashcards system to practice spanish conjugation effectively


# Usage

Online demo:


Local test

    cd spanish-cards
    # python3
    python -m http.server 8000
    # or python2
    python -m SimpleHTTPServer 8000

Then go to <http://localhost:8000>
